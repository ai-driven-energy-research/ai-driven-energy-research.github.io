The AIDER Manifesto

Academic publishing is broken. Papers sit behind paywalls. Results cannot be reproduced. Code is never shared. The process of research is hidden. AIDER exists to fix this.

"Open process, not just open access." We don't just share the paper. We share how it was made — every AI session, every human decision, every iteration.
"AI is a collaborator, not a secret." Using AI to write, code, or analyse is encouraged and must be disclosed. There is no shame in working with AI — it is the future of research.
"If it can't be reproduced, it doesn't count." Code must run. Data must be available. Results must match. This is not optional.
"A paper should be a living product." Published papers live on GitHub. They can be forked, improved, and extended by anyone.
"Free as in freedom, free as in beer." No paywalls. No author charges. No money changes hands. Ever. Science belongs to everyone.

How to Submit

What You Must Provide

Every submission to AIDER includes five components — the paper is only one of them:

  • Paper manuscript — LaTeX or Markdown
  • Code repository — all source code to reproduce every result, with setup instructions and dependency management
  • Data — included in the repo or hosted on Zenodo / HuggingFace with a DOI
  • Research process log — AI agent logs, prompts, session transcripts, and a timestamped record of all human decisions
  • Reproducibility checklist — confirming code runs, data is available, and results match

Review Process

All reviews are open and signed — reviewer names are published alongside their reviews. We believe accountability produces better science.

Two-Track Review

  • Track A: Scientific merit — Is the research sound, novel, and well-presented?
  • Track B: Reproducibility — Can a reviewer clone the repo, run the code, and get the claimed results? This is a hard gate. If the code does not run, the paper is not accepted.

AI-assisted review is allowed and encouraged — reviewers must disclose their use of AI tools. Review artifacts are published alongside the paper.
